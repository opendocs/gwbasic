GW-BASIC Interpreter and Manual

This repository archives GW-BASIC 3.23 interpreter and its manual.

 

The interpreter and documentation in this repository have been obtained from KindlyRat's GeoCities page archived here. The HTML files in the documentation have been edited just a little bit to add an HTML <meta> tag to specify the character set as ISO-8559-1. This was necessary to render the documents properly via GitHub Pages which always serves pages with a default encoding of UTF-8.
